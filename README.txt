jyn ELITE

Ajouts :
- Logo J dans l'icône et en haut du bandeau
- Nom de l'app : jyn
- Variétés de repas
- Plusieurs séances / exercices
- Notifications eau + sport
- Compteur streak
- Sans widget Android
- Sans mode discipline hardcore
- Sans version Play Store

Installation :
1. Va sur https://app.netlify.com/drop
2. Dépose ce fichier ZIP
3. Ouvre l'URL HTTPS générée sur Android avec Chrome
4. Appuie sur "Installer l’application"

Important :
Les rappels web restent surtout fiables quand l'app est ouverte.
